# FakturaBot Server Agent Context Example

Use this file as a safe template for a local-only server context document.

Keep the real working copy outside the public index and store only non-secret structure here.

Suggested sections:
- purpose
- current deployment stage
- project separation rules
- local server context
- private runbook area
- restart and verification checklist

Do not put into the public repository:
- real hostnames or IPs
- SSH commands with real targets
- absolute server paths
- secrets or tokens
- private operational notes
